## Add Staff

You have **two options** to add new staff - coaches - through an <a href="https://performancemetrics.atlassian.net/wiki/spaces/FF/pages/163844/User+Manual#Invite-link" rel="nofollow"><strong>invite link</strong></a> or <a href="https://performancemetrics.atlassian.net/wiki/spaces/FF/pages/163844/User+Manual#Manual-Creation" rel="nofollow"><strong>manual creation</strong></a>.

### Invite link

1.  Click the blue button on the top left corner saying **Invite coach**

![image-20251120-184124.png](6821c87dc7ed2de8adfaff68c86da4c133e6079911e10f379a524af0f37fcc63)

2.  You will be redirected to the Invite Links page, where you should click the **Create invite link** button on the top left corner

![image-20251121-113958.png](e442132cab1d14069db930c86598f0df7d413340b8d55a1a85436e8d0af06c14)

3.  Fill in the required fields

![image-20251121-114234.png](1da2d0a07ed021cdc122745529a651501d64608023f600886b145a0aa0c4f056)

4.  Copy the invite link and send it to the new user.

![Screenshot 2025-11-21 175921.png](87827c93412a5bf90e4cebc8005b7cc2e1bd2e12a23fc2dcb169e0e4b2f5dba9)

> **Info:** You can **send it through any platform** (e.g. WhatsApp, messenger, SMS, etc.)

5.  You can manage the invite link in the Invite Links page.

![Screenshot 2025-11-21 180833.png](9365b0bce51fb37ad8bb0e23cbb2c2b985a5a9e911991c440b974b16148724c3)

### Manual Creation

> **Info:** This **only works** if you want to create a coach that already is a **registered** **user** on the platform.

*Example: If you are an **admin and you also want to have a coach profile**, you can **add yourself as a coach** with this method**.***

1.  Click on the **button with an icon** on **top right** corner

![image-20251121-185531.png](5d8bf337deb50277da97a54bd3bd296176c50db416a52ec4ffc55b4c28df07ae)

2.  Fill in the **required fields** and **click create coach**.

![image-20251121-185911.png](5dc9f21c150f3c062afc6768ea34fec93fecb6f8608ff63e89e5b39b04929313)

3.  You will be able to manage all registered coach in the Staff page

![image-20251121-190040.png](9ae43462d4940762b720a55838ae5c4d2d524f3d82a4fcda87ac42087959c1d9)

## Add Athletes

You have **two options** to add new athletes - through an <a href="https://performancemetrics.atlassian.net/wiki/spaces/FF/pages/163844/User+Manual#Invite-Link" rel="nofollow"><strong>invite link</strong></a> or <a href="https://performancemetrics.atlassian.net/wiki/spaces/FF/pages/163844/User+Manual#Manual-Creation" rel="nofollow"><strong>manual creation</strong></a>.

### Invite Link

1.  Navigate to the Athlete tab and click the blue Invite athlete button on the top right corner.

![image-20251121-192549.png](b60d39e3ed6df80a6bc49dbc04d731e4f3f491fedc11af01ecf5afe39f2151c5)

2.  You will be redirected to the Invite Links page, where you should click the **Create invite link button** on the **top left corner.**

![image-20251121-113958.png](e442132cab1d14069db930c86598f0df7d413340b8d55a1a85436e8d0af06c14)

3.  Fill in the **required fields** and click on the **create invite link button**

![image-20251121-194227.png](f0541a0efb9f221b82ebc4f41f840de695b0e31f78d39852abe2a920fd2a0557)

4.  **Copy** the invite link and **send it** to the new user.

![Screenshot 2025-11-21 175921.png](87827c93412a5bf90e4cebc8005b7cc2e1bd2e12a23fc2dcb169e0e4b2f5dba9)

> **Info:** You can **send it through any platform** (e.g. WhatsApp, messenger, SMS, etc.)

5.  You can **manage** the invite link in the **Invite Links page.**

![image-20251121-195242.png](1570b84ed3d7c0c5f5025433c4a909021101a528c3216901bdfe7b4ad3356a67)

### Manual Creation

> **Info:** This **only works** if you want to create an athlete that already is a **registered** **user** on the platform.

*Example: If you are an **admin and you also want to have an athlete profile**, you can **add yourself as an athlete** with this method**.***

1.  Click on the **button with an icon** on **top right** corner.

2.  Fill in the **required fields** and click on the **Create Athlete button**

![image-20251121-202803.png](69f39b1c9918828de97968bb1316f96b41fa547da8d2bcc0ff608ec1b8e83666)

4.  Success! You will be able to manage any athlete profile in the Athlete page.

## Scheduling

In the Scheduling page you can create the sessions that athletes will have (training sessions / match day).

> **Note:** You **must create these sessions** for **athletes** to be able to **submit their RPE.**

1.  Click the **Scheduling tab** on the **nav bar on the left**

![image-20251122-155638.png](a4d298d9736e38862552e2ad68a6bca9f4ee651346e0812c43bfc488eadd0c99)

2.  To create a new session **click the New Session black button** on the **top right corner** of the page.

![image-20251122-160206.png](458cf4be50cc6ccd97ed49f5a56471529ca651dc3a717fb017bcba8808a9f91c)

3.  Fill in the **required fields** - select the **session type** (training session, match day), **select the participants**, scroll down, and click the **create section button.**

![image-20251122-155959.png](f5622eb4e22c516820a959fa6693bf2fbda7792d534a3b0ad78828ae68ee8b60)

> **Info:** If you want to **select all athletes**, you can click the **square on the left of Athlete Name** in the **Participants table.**

> **Note:** You can **adjust the session duration for each athlete** after the session by **editing the session** and changing the duration value in the **Participants table.**

4.  You can **manage the training sessions** in the **Scheduling page.**

![image-20251122-184828.png](380df78cce38354e57070a12b9a61d6c2e6fbe7bfdf98ef50bd73ade5652319f)

## Response Monitoring

In the Response Monitoring page you will be able to analyse the volume of responses to the RPE & Wellness questionnaires.

The pages show the aggregated results for the team, and also a detailed view for each athlete. By clicking each card you will be able to manage individual responses.

1.  **Click the Response Monitoring tab in the the nav bar on the left.**

![image-20251122-185427.png](fade79af8901da5d78463fdc74caa58f773a6c475fc0e9281ec1cb9c21a73938)

2.  If you **click the RPE Responses Card** you will open a Dialog box where you can se which athlete responded or not, and manage the responses the responses of each athlete.

![image-20251122-190605.png](b27b40236d677b773cb6dfecadbebda910302585238e16bb15d2315eafa10717)![image-20251122-190153.png](e11cf2c1bc5b53687c099f81fcf8598849aeba7915b281abe23e5a676815e762)

3.  To **manage the RPE Submissions** just click the **Manage RPE Submission button** above the table.

![image-20251122-190725.png](e2f980db62a70018ec6bc64078a9a5f9327a28b70a291f1669cdad4950365afa)

4.  **Click the RPE cell** in the table to **edit a value** or **add a value to a empty cell.** You can also **add notes to the training session of each athlete** - these will **only be visible to you.**

![image-20251122-191322.png](3824de0c524c09cc78ba3eec74f0c727d6c4268a38a1bb62ec4bc7df604e2062)
